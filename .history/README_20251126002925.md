# Asset Management System

A full-stack web application for managing IT assets and support tickets with role-based access control.

## Features

### Completed Features
- User authentication with role-based access (Admin/Normal User)
- ✅ Asset management (CRUD operations)
- ✅ Asset detail view page
- ✅ Support ticket system
- ✅ Dashboard with statistics and charts
- ✅ Excel export for assets and tickets
- ✅ Search and filter functionality
- ✅ Responsive UI design
- ✅ Real-time data updates

## Technologies Used

### Backend
- C# .NET 8 (Minimal API)
- Entity Framework Core
- SQL Server
- ClosedXML (Excel generation)

### Frontend
- Vue.js 3 (Composition API)
- Vue Router
- Axios
- Chart.js (Data visualization)
- Pinia (State management)

## Setup Instructions

### Prerequisites
- .NET 8 SDK
- SQL Server or SQL Server Express
- Node.js (v18+)

### Backend Setup

1. Navigate to backend folder
2. Update connection string in `appsettings.json`:
   ```json
   "ConnectionStrings": {
     "DefaultConnection": "Server=localhost\\SQLEXPRESS;Database=AssetDB;Integrated Security=True;TrustServerCertificate=True;"
   }
   ```
3. Create database:
   ```bash
   dotnet ef migrations add InitialCreate
   dotnet ef database update
   ```
4. Run the API:
   ```bash
   dotnet run
   ```

### Frontend Setup

1. Navigate to frontend folder
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create `.env` file:
   ```env
   VITE_API_BASE_URL=http://localhost:5011
   ```
4. Run the application:
   ```bash
   npm run dev
   ```
