# AssetSolution

AssetSolution is an Asset Management Project that includes a backend (AssetAPI) and a frontend (AssetFrontend).  
This project is built using **.NET** for the API and **Vue 3 + Vite** for the frontend.

## Prerequisites

Make sure you have installed:

- [.NET 6 SDK or later](https://dotnet.microsoft.com/download)  
- [Node.js](https://nodejs.org/) (v16 or later)  
- npm (comes with Node.js)  
- Optional: Visual Studio or VS Code

```bash
